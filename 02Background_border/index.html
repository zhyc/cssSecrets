半透明边框

半透明边框需要加：

background-cilp: padding-box

多重边框

多重边框方案一 box-shadow：

box-shadow : 0 0 0 10px #655, 0 0 0 20px deeppink

box-shadow支持逗号分隔，可以创建任意数量的边框，可做圆角边框

注意： 不影响布局，不受 box-sizing的影响

多重边框方案二 outline：

border: 10px solid #665555; outline: 5px solid deeppink;

注意： border-radius不切合

设置 outlin-offset负值可做缝线效果

灵活的背景定位

背景定位方案一 background-position：

background-position: right 20px bottom 10px; 可设置距离右和下的距离

背景定位方案二 background-origin：

background-origin: content-box;padding: 10px; 背景偏移量可与padding值一致

背景定位方案三 calc()：

background-position: calc(100% - 20px) calc(100% - 10px); 可计算边距

使用cale()在 + 、 -两侧需要加空白符，否则产出解析错误
